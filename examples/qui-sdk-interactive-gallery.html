QuiBrowserSDK - Interactive Gallery

Features:

  • 3D image gallery
  • Hand menu (palm gesture)
  • Gesture recognition
  • Voice commands (100+ languages)
  • Gaze interaction
  • Developer tools (FPS, memory)

Gestures:

  • Palm: Show menu
  • Pinch: Select
  • Point: Highlight
  • Fist: Close

Voice:

  • "next" - Next image
  • "previous" - Previous image
  • "menu" - Toggle menu
  • "help" - Show help
