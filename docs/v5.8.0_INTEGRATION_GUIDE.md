# v5.8.0 Integration Guide

## Quick Integration (3 Steps)

### Step 1: Load Modules
```html
<script src='assets/js/vr-wasm-simd-accelerator.js'></script>
<script src='assets/js/vr-advanced-gesture-dataset.js'></script>
<script src='assets/js/vr-gesture-customization.js'></script>
<script src='assets/js/vr-performance-dashboard.js'></script>
```

### Step 2: Initialize Systems
```javascript
// SIMD Acceleration
const accelerator = new VRWasmSimdAccelerator();
await accelerator.initialize();
await accelerator.optimizeGestureRecognition(gestureEngine);

// Gesture Dataset
const gestures = new VRAdvancedGestureDataset();

// Customization System
const customization = new VRGestureCustomization(gestureEngine);
await customization.initialize();

// Performance Dashboard
const dashboard = new VRPerformanceDashboard(profiler);
await dashboard.initialize();
```

### Step 3: Use Features
```javascript
// Get gestures
const indexPoint = gestures.getGesture('index-point');
const controls = gestures.getFamily('control');

// Record macro
customization.startMacroRecording('combo');
// User performs gestures
customization.stopMacroRecording();

// Create profile
customization.createProfile('gaming');
customization.bindGestureToAction('index-point', 'jump');
```

## Example: Gaming Profile

```javascript
// Create gaming profile
customization.createProfile('gaming');
customization.switchProfile('gaming');

// Bind gestures
customization.bindGestureToAction('index-point', 'jump');
customization.bindGestureToAction('peace-sign', 'attack');
customization.bindGestureToAction('fist-closed', 'defend');

// Record combo macro
customization.startMacroRecording('combo-attack');
// User performs: point -> peace-sign -> punch
customization.stopMacroRecording();

// Execute on trigger
customization.bindGestureToAction('combo-attack', 'ultimate-attack');
```

## Troubleshooting

```javascript
// Check SIMD support
const caps = accelerator.getCapabilities();
console.log('SIMD:', caps.simdSupported);
console.log('WASM:', caps.wasmSupported);

// Check gestures
const gesture = gestures.getGesture('index-point');
console.log('Confidence:', gesture.confidence);

// Check macros
const status = customization.getRecordingStatus();
console.log('Recording:', status.recording);
```
