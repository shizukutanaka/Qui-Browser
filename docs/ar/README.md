# نظرة عامة على متصفح Qui (العربية)

## رؤية المنتج

متصفح Qui هو متصفح ويب خفيف موجه لتجارب الواقع الافتراضي، يدعم امتدادات Chrome،
ويحسن بث الفيديو، ويطبق التحكم في الاشتراكات عبر Stripe.

## الميزات الرئيسية

- **دعم الامتدادات**: واجهة `/api/extensions` لتثبيت الامتدادات المتوافقة مع
  Chrome وتحديثها وإزالتها.
- **خط أنابيب الوسائط**: الملف `utils/media-pipeline.js` يعالج طلبات النطاق
  لتوفير تشغيل سلس في الواقع الافتراضي.
- **ضبط الاشتراك**: الملف `server-lightweight.js` يتحقق من حالة اشتراك Stripe
  قبل إتاحة الميزات المميزة.

## البدء السريع

1. تثبيت الاعتمادات: `npm install`
2. نسخ `.env.example` إلى `.env` وإدخال مفاتيح Stripe.
3. تشغيل الخادم الخفيف: `npm start`
4. فتح واجهة الواقع الافتراضي على النظارة أو المحاكي.

## دليل المجلدات

- `server-lightweight.js`: الخادم الأساسي ومسارات REST.
- `utils/stripe-service.js`: عمليات Stripe Checkout ومعالجة Webhook.
- `extensions/manager.js`: إدارة دائمة للامتدادات المثبتة.
- `docs/`: حزمة الوثائق متعددة اللغات.

## الدعم والتغذية الراجعة

أرسل التقارير والمقترحات عبر متتبع المشروع، مع مراعاة خارطة الطريق الموجهة
للواقع الافتراضي.
