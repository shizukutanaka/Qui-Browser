# Qui ब्राउज़र परिचय (हिन्दी)

## उत्पाद दृष्टि

Qui ब्राउज़र एक VR-केंद्रित हल्का वेब अनुभव प्रदान करता है। यह Chrome एक्सटेंशन
समर्थन, वीडियो स्ट्रीमिंग अनुकूलन और Stripe आधारित सदस्यता नियंत्रण को एकीकृत
करता है।

## मुख्य क्षमताएँ

- **एक्सटेंशन अनुकूलता**: `/api/extensions` API के माध्यम से Chrome संगत
  एक्सटेंशन स्थापित, अद्यतन और हटाएँ।
- **मीडिया पाइपलाइन**: `utils/media-pipeline.js` रेंज अनुरोधों को संभालता है,
  जिससे VR में सुचारु वीडियो प्लेबैक संभव होता है।
- **सदस्यता नियंत्रण**: `server-lightweight.js` प्रीमियम सुविधाएँ देने से पहले
  Stripe सदस्यता स्थिति की जाँच करता है।

## त्वरित शुरुआत

1. निर्भरताएँ स्थापित करें: `npm install`
2. `.env.example` को `.env` में कॉपी कर Stripe कुंजी दर्ज करें।
3. हल्का सर्वर प्रारंभ करें: `npm start`
4. VR हेडसेट या एमुलेटर में UI खोले।

## निर्देशिका मार्गदर्शिका

- `server-lightweight.js`: मुख्य HTTP सर्वर और रूटिंग लॉजिक।
- `utils/stripe-service.js`: Stripe Checkout और Webhook हैंडलिंग।
- `extensions/manager.js`: स्थापित एक्सटेंशन का स्थायी प्रबंधन।
- `docs/`: बहुभाषी दस्तावेज़ संग्रह।

## सहायता और प्रतिपुष्टि

प्रोजेक्ट ट्रैकर पर समस्याएँ या सुझाव दर्ज करें और VR-केंद्रित रोडमैप के अनुरूप
प्राथमिकता तय करें।
