# Обзор Qui Browser (Русский)

## Видение продукта

Qui Browser — лёгкий браузер, ориентированный на VR, поддерживает расширения
Chrome, оптимизирован для потокового видео и использует Stripe для управления
подписками.

## Ключевые возможности

- **Поддержка расширений**: API `/api/extensions` для установки, обновления и
  удаления расширений, совместимых с Chrome.
- **Медиапоток**: `utils/media-pipeline.js` обрабатывает диапазонные запросы,
  обеспечивая плавное воспроизведение в VR.
- **Контроль подписки**: `server-lightweight.js` проверяет статус Stripe перед
  выдачей премиальных функций.

## Быстрый старт

1. Установите зависимости: `npm install`
2. Скопируйте `.env.example` в `.env` и заполните ключи Stripe.
3. Запустите облегчённый сервер: `npm start`
4. Откройте VR-интерфейс на гарнитуре или эмуляторе.

## Структура каталогов

- `server-lightweight.js`: основной HTTP-сервер и маршрутизация.
- `utils/stripe-service.js`: обработка Stripe Checkout и вебхуков.
- `extensions/manager.js`: постоянное хранение установленных расширений.
- `docs/`: документация на разных языках.
