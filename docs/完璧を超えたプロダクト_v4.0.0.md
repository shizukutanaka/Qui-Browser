**使用例:**
```javascript
const screenReader = new VRAIScreenReader({
  detectionInterval: 100, // 10 Hz (VRSight標準)

await screenReader.initialize(scene, camera, xrSession);
```

### 3. 協調ブラウジング (WebRTC + CRDT)

**ファイル:** `assets/js/vr-collaborative-browsing.js` (900行以上)

**リアルタイムマルチユーザーVRブラウジング:**
- WebRTCによるピアツーピア（P2P）接続
- CRDT（競合のないレプリケートデータ型）による状態同期
- **2-11名の同時ユーザー**サポート（Quest ネットワーク研究に基づく）
- リアルタイムアバター同期（20 Hz更新レート）
- 空間音声チャット（3D位置音声）
- 共有ブックマークとアノテーション
- **最小レイテンシ:** P2P <50ms、リレー <100ms

**CRDT の利点:**
- 中央サーバー不要
- 自動競合解決
- 最終的一貫性（すべてのピアが同じ状態に収束）
- オフライン動作、後で同期

**研究基盤:**
- VRにおける初のCRDT実装（2025年3月、IEEE）
- LWW-Element-Set（Last-Write-Wins）
- OR-Set（Observed-Remove Set）
- 因果関係追跡のためのベクトルクロック
