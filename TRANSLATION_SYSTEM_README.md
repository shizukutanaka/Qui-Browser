## 🚀 使用方法

### 基本操作

1. **言語パネルを開く**: ナビゲーションバーの🌐ボタンをクリック
2. **言語を選択**: 120言語以上から選択可能
3. **高度な機能**: タブで音声・OCR・会議機能を切り替え

### キーボードショートカット

| ショートカット | 機能 |
|---------------|------|
| `Ctrl+Shift+L` | 言語パネルを開く |
| `Ctrl+Shift+T` | 音声翻訳の開始/停止 |
| `Ctrl+Shift+O` | OCRモードの切り替え |
| `Ctrl+Shift+C` | 会議モードの開始/停止 |
| `Ctrl+Shift+D` | 翻訳ダッシュボード表示 |
| `Ctrl+Shift+A` | すべての高度な機能を有効化 |
| `Ctrl+Shift+U` | 究極翻訳（選択テキスト） |

### 右クリックメニュー

テキストを選択して右クリックすると、翻訳オプションが表示されます：
- 選択テキストの翻訳
- ページ全体の翻訳
- 翻訳結果のコピー

const translation = await window.translate('Hello world', {
    sourceLang: 'en',
    targetLang: 'ja'

// 次世代AI翻訳

// 音声翻訳

    targetLang: 'ja'
});

// 会議翻訳

    platform: 'zoom',
    features: ['subtitles', 'chat', 'audio']
});
