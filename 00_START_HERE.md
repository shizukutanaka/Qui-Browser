## ğŸš€ Deployment Options

### Option 1: GitHub Pages (Easiest)
```bash

# Automatically deploys via GitHub Actions
```


```bash

```

```bash
# Runs on http://localhost:8080
```

```bash
# Copy dist/ to any web server
scp -r dist/* user@host:/var/www/html
```
